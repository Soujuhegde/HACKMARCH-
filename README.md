# InsightCare — AI for Longevity 💗

**InsightCare** is a cutting-edge, data-driven health platform designed to help users understand their "real" biological age and take actionable steps toward a longer, healthier life. By combining advanced clinical methodologies with state-of-the-art AI, InsightCare transforms complex biomarkers into clear, personalized longevity strategies.

---

## 🚀 Key Features

### 🧬 Biological Age Prediction
Experience the power of the **PhenoAge methodology**. InsightCare analyzes key health markers to estimate your biological age, providing a more accurate reflection of your health than the calendar alone.

### 🤖 Personalized AI Assistant
Meet your dedicated longevity coach. Our **AI Health Assistant** is context-aware, meaning it knows your specific health data and provides tailored advice on nutrition, exercise, and lifestyle improvements without the generic fluff.

### 📊 Dynamic Risk Assessment
Visualize your health across three critical pillars:
- **Cardiovascular Health**
- **Metabolic Function**
- **Cognitive Resilience**
Real-time, color-coded feedback helps you identify where you're thriving and where you need focus.

### 🔮 Future Health Projections
Interactive simulations allow you to see into the future. Adjust your habits—like sleep, steps, or stress—and watch in real-time as your projected life expectancy and biological age improve.

### 🛡️ Clinical Data Validation
Built with reliability at its core. Our **Validation Engine** includes:
- **Hard Guardrails**: Prevents unrealistic data entry.
- **Clinical Warnings**: Detects medical risks like Hypertension or Tachycardia.
- **Confidence Scoring**: A transparent score (0–100%) indicating the reliability of your health predictions.

---

## 🛠️ Technology Stack

- **Frontend**: [Streamlit](https://streamlit.io/) for a fast, responsive, and modern user interface.
- **AI Core**: Powered by **Google Gemini Pro** (Generative AI) for personalized health consultations.
- **Visualization**: [Plotly](https://plotly.com/) for interactive charts and longevity projections.
- **Data Engine**: Pandas and NumPy for complex biomarker analysis and risk scoring.

## 💻 Getting Started

### Prerequisites
- Python 3.9 or higher
- A Google Gemini API Key

### Installation
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Soujuhegde/HACKMARCH-.git
    cd HACKMARCH-
    ```

2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

3.  **Configure Environment**:
    Create a `.env` file or set up your secret in Streamlit:
    ```env
    GOOGLE_API_KEY = "your-api-key-here"
    ```

4.  **Run the Application**:
    ```bash
    streamlit run app.py
    ```
